Joy Game Grid

Challenge yourself with this classic number puzzle game. Arrange the numbers in order by sliding tiles into the empty space.

Moves: 0

How to Play

1. Game Objective

Arrange the numbers 1-8 in ascending order, with the empty space at the bottom right.

2. Moving Tiles

Click or tap any number adjacent to the empty space to slide it into the empty position.

3. Winning the Game

Complete the puzzle by arranging all numbers in order: 1-2-3 in the first row, 4-5-6 in the second row, and 7-8-empty in the last row.

Game Features

Simple Controls

Works perfectly on both desktop and mobile devices with touch support.

Move Counter

Track your progress and try to solve the puzzle in fewer moves.

Instant Reset

Start a new game at any time with the "New Game" button.
